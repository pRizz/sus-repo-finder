About Sus Repo Finder

Sus Repo Finder scans Rust crates for potentially suspicious patterns in build scripts and procedural macros. These patterns might indicate security risks such as:

  • Network calls during build
  • Accessing sensitive paths
  • Shell command execution
  • Environment variable access
  • Unsafe code blocks
  • Obfuscated code patterns

Note: Not all findings indicate malicious intent. Many legitimate crates have valid reasons for these patterns. Always review findings in context.
